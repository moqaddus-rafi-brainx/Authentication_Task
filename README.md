# Database Integration Branch

This branch focuses on the **database setup** and **user authentication model** using **MongoDB** and **Mongoose** in a Node.js environment.

## 🔧 Features Implemented

- ✅ Integrated **MongoDB** using **Mongoose**
- ✅ Created **User Authentication Model** for handling user data
- ✅ Structured database connection logic with proper error handling

## 🛠️ Technologies Used

- **Node.js Expressjs**
- **MongoDB**
- **Mongoose** (MongoDB ODM for Node.js)
